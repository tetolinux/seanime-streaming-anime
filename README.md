# Seanime Animepahe Streaming Provider

An external media streaming provider extension for Seanime built specifically to scrape and stream links from `animepahe.ch`.

## Features
- Full Title Search support matching Alternative titles.
- Native Sub/Dub classification processing.
- Multi-server decoding integration support (Blogger, OK.ru, Megaplay).
- Base64 payload scraping extraction engine bypass.
